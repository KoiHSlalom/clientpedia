## Consultant Profile — Template

Purpose: concise profile used for internal staffing and role-matching.

Fields (required)
- Internal ID
- Full name
- Primary skills (tags)
- Seniority / Grade
- Primary location / timezone
- Current availability (dates or %)
- Internal manager

Fields (optional)
- Resume / CV link
- Hourly or day rate or grade
- Contact (email, phone)
- Calendar link (optional)
- Notes (recent projects, certifications)

Privacy
- Mark visibility restrictions for client-restricted opportunities.

Example
- ID: I-100
- Name: Alex Smith
- Skills: AWS, Terraform, Data Engineering
- Seniority: Senior
- Availability: 2026-06-01 (50% available)
