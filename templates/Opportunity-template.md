## Opportunity / Role — Template

Purpose: capture upcoming projects, roles, and staffing needs linked to a client.

Fields (required)
- Opportunity ID
- Client ID / Name
- Title
- Type (project / pilot / advisory)
- Stage (e.g., Prospect, Qualified, Proposal, Won)
- Expected start date
- Estimated duration (days/weeks/months)
- Estimated value (optional)
- Owner (person/team)

Role definitions (per opportunity)
- Role ID
- Role title
- Skills / tags
- Seniority
- Headcount
- Location / Remote
- Estimated allocation (FTE%)

Links & notes
- Linked CRM ID
- Related documents (SOW, MSA, proposal)
- Privacy / restriction flags

Example
- Opportunity ID: O-2026-01
- Client: Acme Corp (C001)
- Title: Cloud Migration – Phase 2
- Stage: Qualified
- Expected start: 2026-05-01
