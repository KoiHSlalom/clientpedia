## Pilot Checklist

Purpose: runnable checklist for the pilot run (8 weeks).

Pre-pilot (Weeks 0–2)
- Select pilot clients (3–5) and owners.
- Define pilot KPIs and measurement plan.
- Create templates and import CSVs.
- Configure SSO and RBAC for pilot users.

Data load (Weeks 2–4)
- Import client profiles, contacts, engagements, opportunities, and consultant profiles.
- Validate search indexing and document attachments.

Validation (Weeks 5–7)
- Execute sample search and relevance tests.
- Run consultant flow: bookmark + express interest tests.
- Verify CRM sync and resolve sync conflicts.

Wrap-up (Week 8)
- Collect feedback, measure KPIs, and prepare rollout plan.
